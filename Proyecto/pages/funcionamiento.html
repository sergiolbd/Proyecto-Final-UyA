Manual de uso

Esta Web te permitirá organizar tu día a día, mediante las opciones de añadir, completar y comprobar tareas pendientes. Para ello debemos de seguir una serie de pasos, dependiendo del punto en el que nos encontremos, pues se distinguirá en usuarios que con anterioridad hayan añadido tareas, y usuarios nuevos.

  • Nuevo usuario
    1. Acceder a la sección lista de tareas pendientes
    2. Añadir una nota, la cual se asocia al usuario indicado
    3. Consultar todas tus tareas pendientes desde la sección Calendario
    4. Posibilidad de acceder a lista de tareas pendientes para poder observar tus tareas y marcar si ya las completaste para que se eliminen
  • Usuario
    • Acceder a la sección calendario y consultar las tareas para un determinado usuario
    • Acceder a la sección lista de tareas pendientes y añadir nuevas tareas o marcar como terminadas otras
